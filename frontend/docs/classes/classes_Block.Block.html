    Class Block

    Classe Block

    Cette classe encapsule un HTMLElement pour offrir une interface orientée objet plus intuitive et fluide lors de la manipulation du DOM.

    Elle hérite d’un Emitter, ce qui lui permet d’émettre et d’écouter des événements personnalisés.

    Example

    const div = new Block('div', { class: 'container' });
    div.setStyle('color', 'red')
       .html('<p>Hello world</p>')
       .appendTo(document.body);

    Remarks

    Block agit comme un wrapper simplifié autour des éléments du DOM tout en ajoutant des fonctionnalités pratiques (gestion des classes, attributs, styles, écouteurs d’événements, etc.).

      Enregistre un nouvel écouteur sur un événement donné.

      Parameters

        Nom de l’événement à écouter.

      • triggerCallback: TListenerTriggerCallback

        Fonction exécutée lorsque l’événement est émis.

      Returns Listener

      L’objet Listener associé à cet abonnement.

      Example

      Remarks

      Le Listener retourné contient un callback off() permettant de se désabonner facilement :

      const l = emitter.on("refresh", fn);
      l.off(); // retire le listener
